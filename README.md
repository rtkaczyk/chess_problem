Chess Problem
=============

## Compiling

```
$ sbt assembly
```

## Running


```
$ ./run <mode> [<problem_set> [java_options ...]]

# where:
#   - mode: either "recursive" or "parallel"
#   - problem_set: 7 numbers decribing the problem ("7 7 2 2 0 2 1" for the main problem, which is default)
```
